github简介及个人静态网站的搭建

github是什么？

  • 首先他是一家著名的互联网公司，由三位开发者在2008年4月创办。迄今拥有59名全职员工。github 主要提供基于git的版本托管服务。

  • github无疑是这个星球上最流行的开源托管服务。目前已托管400多万git项目，不仅越来越多知名开源项目迁入GitHub，比如Ruby、jQuery、Erlang/OTP；近三年流行的开源库往往在GitHub首发，例如：BootStrap、Node.js等。

  • github号称是程序猿界的Facebook，他拥有140多万的开源社区，其中有顶级黑客Linux发明者，还有各种Geek，那么如何提高自己呢？

理想的环境Git+Github+Markdown

  • Markdown 是一种「电子邮件」风格的「标记语言」。

  • 可以更专注文字而不是排版，纯文本可读性极好

  • 格式转换方便，生成html、pdf等

  • Markdown语法简介

搭建一个静态站点

  • 在github上为每个用户以及每个项目都可以建立个人主页或项目主页

  • 创建github账号，新建repository，名字格式为：你的账号.github.com

  • git clone 仓库地址,创建index.html并提交到github
